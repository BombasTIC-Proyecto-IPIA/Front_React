## Instalación
***
1. Clonar el proyecto 
```
git clone https://github.com/La-Volta/la-volta-client.git
```
2. Hacer el  siguiente comando para instalar las dependencias de react
```
npm i
```
3. Hacer el  siguiente comando para inciar 
```
npm run dev 
```
